# MongoDB_Shell_Task_Management

In these assignment we have created a Database name Task_Management. We have created three collections named users, tasks and assignedTasks.
Following are the operations performed on Task_Management Database-
1-Create database
2- create collections
3- insert users into users collection
4- insert tasks into task collections
5- insert assigned task into assigned task collections
6- CRUD-Operations - a- Create a user
                     b- Update a user
                     c- Delete a user
                     d- Create a task
                     e- Read all task for specific user
                     f- Read a task by Id
                     g- Update a task
                     h- Delete a task
                     i- Get task by priority
                     j- Get task by status
                     k- Assign task
                     l- Get all asignments
                     m- Remove assignments
